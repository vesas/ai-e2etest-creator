# Customers Table

The application displays a table listing customers.

## Table Columns

- **ID** - Unique identifier for each customer
- **Name** - Full name of the customer
- **Email** - Email address of the customer
- **Phone** - Phone number of the customer
- **Status** - Current status of the customer (Active, Inactive)
- **Created At** - Date the customer was created
- **Country** - Country where the customer is located

## Behavior

- The table is displayed on the main page when the app loads
- All customers are listed in the table
- Customers are sorted by ID in ascending order by default
